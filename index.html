Features

  • Sensitivity Multiplier: A parameter that multiplies the output vector.
  • Gain Switch: Determines the application of the acceleration graph shape.
  • Offsets: Defines a speed threshold for acceleration to engage.
  • Caps: Prevents acceleration after a certain point.
  • Anisotropy: Sensitivity settings based on horizontal and vertical movement characteristics.

Acceleration Styles

Linear

The simplest and most user-friendly style.

Classic

A style found in games like Quake.

Power

A style found in CS:GO, allowing for fine adjustments.

Natural

Provides smooth transitions starting from 1.
